Create TAP device with Windows device manager. 
	- Add legacy hardware (network adapter) from TAP-Windows Provider V9	
	- Rename it to "tap0" in the Network Connections control panel 

emulator -avd Pixel_5_API_29 -qemu -device virtio-net-pci,netdev=eth1 -netdev tap,id=eth1,ifname=tap0,script=no

should make the tap0 device appear connected in Windows Network Connections control panel 

adb shell
$ su
# ip link show 
Should show eth1; if not: reboot may help
# ip link set dev eth1 up
# ip addr add 192.168.178.2/24 dev eth1
